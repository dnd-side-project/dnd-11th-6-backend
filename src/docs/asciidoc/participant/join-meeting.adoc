===== `Authorization Header 처리:`
- Authorization Header는 필수값이 아닙니다.
- 요청 시, Authorization 헤더는 Bearer accessToken값 형식으로 전달해주세요.
- 단, accessToken 값이 없는 경우(사용자가 처음 모임에 참여하는 경우) Authorization 헤더를 포함하지 마세요.
- 기한이 만료된 accessToken도 가능합니다.

===== `두개 이상의 모임 참여 시 Authorization Header 처리`
- 사용자가 두 개 이상의 모임에 참여할 경우, 헤더에 해당 모임에서 받은 accessToken 값을 반드시 포함해주세요.
- 만약 두 개 이상의 모임에 참여하면서 헤더에 accessToken 값을 전달하지 않으면, 사용자는 이전에 참여한 모임에 더 이상 접근할 수 없습니다.
