==== 모임 상태 설명
- `INACTIVE 비활성화`: 모임 시작일 이전 상태.
- `ACTIVE 활성화`: 모임이 시작된 후 종료일 + 7일까지의 상태.
- `INVALID 유효하지 않음`: 종료일 + 7일이 지난 후의 상태.
