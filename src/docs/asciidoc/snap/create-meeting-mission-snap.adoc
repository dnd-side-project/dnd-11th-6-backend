=== 모임 미션 snap 등록
==== 성공
===== 요청 형식

요청은 `multipart/form-data` 형식을 사용하여 JSON 데이터와 파일을 함께 전송합니다.

- **`name=snap`**: snap 정보를 포함하는 JSON 데이터입니다.

- **`name=image`**: 업로드할 이미지 파일.
